# Dotted — 프로젝트 개요

> 어떤 사진이든 3D 포인트클라우드로

일반 핸드폰으로 찍은 사진을 업로드하면 AI 깊이 추정을 통해
3D 포인트클라우드로 변환해주는 오픈소스 뷰어.

## 핵심 컨셉

- 어떤 사진이든 업로드 → AI 깊이 추정 → 포인트클라우드 → 3D 뷰어
- GPS 유무 상관없이 동작 (실내/실외 모두)
- 메쉬 생성 없이 포인트클라우드까지만 구현
- 오픈소스 포트폴리오

## 확장 계획 (선택적)

- 여러 사진을 하나의 공간으로 묶기 (SfM)
- GPS 있는 사진은 지도 위에 위치 표시
- 커뮤니티 기여 방식 오픈

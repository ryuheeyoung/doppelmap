# Dotted — 프로젝트 개요

> 기억하고 싶은 모든 공간을 3D로

## 한 줄 컨셉

**현실이든 가상이든, 사진 한 장이면 3D 포인트클라우드로.**

## 배경

우리는 다양한 공간에서 기억을 만든다.

- 여행지에서 찍은 경복궁, 동대문
- 마비노기·원스휴먼 하우징에서 꾸민 추억의 집
- 핸드폰으로 찍은 동네 골목, 실내 공간

이 모든 공간은 지금 2D 사진 속에 갇혀있다.
Dotted는 그 사진을 3D 포인트클라우드로 꺼내준다.

## 핵심 컨셉

- **현실 공간** — 핸드폰 사진 (GPS 유무 상관없음, 실내/실외 모두)
- **가상 공간** — 게임 스크린샷, 인게임 하우징 등
- 사진 업로드 → AI 깊이 추정 → 3D 포인트클라우드 뷰어
- 메쉬 생성 없이 포인트클라우드까지 구현
- 오픈소스 포트폴리오

## 확장 계획 (선택적)

- 여러 사진을 하나의 공간으로 합치기 (SfM)
- GPS 있는 사진은 지도 위에 위치 표시
- 커뮤니티 기여 방식 오픈

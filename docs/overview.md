# Doppelmap — 프로젝트 개요

> 현실 공간의 디지털 복제 (Doppel + Map)

한국의 공간을 커뮤니티 사진으로 포인트클라우드화하는 오픈소스 디지털트윈 뷰어.
역사와 현대가 공존하는 공간(동대문역사문화공원, 경복궁 등)을 대상으로 하며,
계절과 시간의 레이어를 함께 표현하는 것을 목표로 함.

## 핵심 컨셉

- 일반 핸드폰 사진/동영상 → GPS + 깊이 추정 → 포인트클라우드
- 메쉬 생성 없이 포인트클라우드 단계까지만 구현 (줌 레벨별 LOD 클러스터링)
- 역사 레이어 / 현재 레이어 토글
- 계절 레이어 전환 (봄/여름/가을/겨울)
- 커뮤니티 기여 방식 (초기엔 관리자만 업로드)
- 오픈소스 포트폴리오

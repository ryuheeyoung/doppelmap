# 기술 스택

## 프론트엔드

| 도구                         | 용도                     |
| ---------------------------- | ------------------------ |
| Next.js 16 (App Router)      | 프레임워크               |
| TypeScript                   | 타입 안전성              |
| Tailwind CSS                 | 스타일링                 |
| Three.js + React Three Fiber | 3D 포인트클라우드 렌더링 |
| MapLibre GL JS               | WebGL 기반 지도          |
